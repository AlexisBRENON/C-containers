~~ Description ~~

Ce petit programme vous permet de tester et de comparer les performance d'une 
table de hachage.

Vous trouverez, dans cette archive (./Resultats), une représentation graphique du temps
de recherche selon le type de rehachage utilisé, et le nombre total d'entrées
(nombre premier (6007) ou non (6000)).

~~ Compilation ~~

Pour compiler :
    ~$ cd BRENON_Alexis
    ~/BRENON_Alexis$ make

~~ Execution ~~

Pour exécuter, une fois la compilation réussie sans erreur, il vous suffit de 
taper la commande suivante :
    ~/BRENON_Alexis$ bin/HashTable

~~ Documentation ~~

Un fichier de génération de documentation est fournis dans le dossier 
BRENON_Alexis/doc.
Après avoir installé doxygen, entrez la commande suivante :
    ~/BRENON_Alexis$ doxygen doc/doxyfile
    
Pour afficher la documentation résultante :
    ~/BRENON_Alexis$ {firefox|opera} doc/html/index.html
